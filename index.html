MortarJs

Strongly and cohesively bond fundamental building blocks of modern JavaScript Web Applications in a jQuery EcoSystem. This mini framework pulls together capabilities such as:

  • Knocjoutjs integration
  • Html fragments
  • Css styling of different formats (css and less)
  • Direct integration with jQuery UI factory.
  • Modular design with requirejs support.

MortarJs will increase consistency in your code base as well as productivity. How?

  • Everything is provided in a clean namespace as well as jQuery plugins.
  • All MortarJs components are designed to provide consistent interfaces.
  • MortarJs will synchronize async operations such as loading html fragments and css as well as binding to knockoutjs view models.
  • Non intrusive design so that all your already existing knockoutjs extensions and jQuery plugins will continue to operate without disrruption.

Performance, modularity and responsiveness are core to MortarJs.
