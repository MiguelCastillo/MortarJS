Github project | Miguel Castillo | Version 0.0.1

To the end user, a Web Application is primarily the means to access information in some shape or form.
To the developer however, a Web Application can be an extremly hostile environment with creative ways to make JavaScript, HTML, and CSS unmanageable. Let MortarJs help you.

Simplicity

with automatic loading css, html fragments, and KnockoutJs integration.

Maintainability

by leveraging modules to encapsulate functionality and data.

Flexibility and reusability

by exploiting jQuery's plugin ecosystem.

Good coding practices and easy to adopt

by leveraging well established toolkits and frameworks such as RequireJs, jQuery, KnockoutJs, Twitter Bootstrap.

Open to everyone

with MIT license. Please contribute!
